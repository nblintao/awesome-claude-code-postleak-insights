# Contributing

Thanks for your interest! Pull Requests and Issues are both welcome.

## What belongs here

- **Must be produced after the 2026-03-31 leak.** Pre-leak articles, independent security research, and earlier reverse-engineering efforts are out of scope.
- **Must focus on Claude Code itself**, not general AI-tool news or unrelated leak events.
- **Must offer substantive analysis** — low-signal news summaries that simply restate the facts don't qualify. We value deep dives into architecture, safety model, workflow design, extension surfaces, and similar topics.
- **No source code mirrors or archives** — this is not a code hosting site.

## Entry format

Each link should follow this format:

```markdown
- [Title](URL) - One sentence on why it's worth reading.
```

Place it under the appropriate section in README.md. If no existing section fits, feel free to propose a new one in your PR.

## How to contribute

1. Fork this repository
2. Add the link to the appropriate section in README.md
3. Submit a Pull Request with a brief note on why the link meets the criteria above
